This is the production hsqldb-2.7.4 with the manifest file updated 

Sealed: false

Allows the addition of the custom Qortal HSQLDBPool and Monitoring Classes

Install the custom jar in maven local repository with the command:

mvn install:install-file \
  -Dfile=lib/org/hsqldb/hsqldb/2.7.4/hsqldb.jar \
  -DgroupId=org.hsqldb \
  -DartifactId=hsqldb \
  -Dversion=2.7.4 \
  -Dclassifier=custom \
  -Dpackaging=jar \
  -DgeneratePom=true
